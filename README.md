# stamp-maker

写真に文字とかざりをのせて、LINEスタンプ用の画像を作れるブラウザツールです。
インストール不要・サーバー不要で、画像はどこにも送信されません(すべてブラウザ内で処理されます)。

**▶ 使ってみる: https://uni-git-maker.github.io/stamp-maker/**

## できること

- 📷 好きな画像に文字をのせる(たて書き / よこ書き、2色分け、サイズ・色変更)
- 💬 あいさつプリセット(「おはよう」「ありがとう」など8種)
- 🐾 かざりスタンプ(にくきゅう・キラキラ)を好きなだけ配置 — ドラッグで移動、ダブルタップで削除
- 🎨 白背景の透過(強さ調整つき)/ 背景色の変更
- 📏 LINEスタンプの比率(370:320)プレビューと余白ガイド表示
- 💾 LINEクリエイターズスタンプの規定サイズで透過PNGを書き出し
  - スタンプ画像: 370×320
  - メイン画像: 240×240
  - トークルームタブ画像: 96×74
  - 高画質PNG: 1024×1024

## 使い方

1. 上のリンクを開く(またはこのリポジトリの `index.html` をブラウザで開く)
2. 「📷 画像を選ぶ」で写真を読み込む
3. 文字・色・かざり・透過を調整する(文字やかざりはドラッグで動かせます)
4. 「保存」の各ボタンで PNG をダウンロードする

スマホ・タブレット・PCのモダンブラウザ(Chrome / Edge / Safari / Firefox)で動作します。

## 開発者向け

ビルド不要の静的サイトです。HTML / CSS / JavaScript のみで構成されています。

```
index.html      画面
css/style.css   スタイル
js/app.js       描画・透過処理・書き出しロジック
```

ローカルで動かす場合は `index.html` をブラウザで開くだけです。
